# RoadTrip Planner

Map your routes, discover hidden gems, and organize every stop along the way. The all-in-one planner for unforgettable road trip adventures.

**[View Live Site](https://roadtrip-plan.netlify.app)**

## About

RoadTrip Planner is a web application that helps users plan, organize, and share road trips. It combines interactive mapping, itinerary building, collaboration features, and travel analytics into one unified platform.

## Features

- **Interactive Route Planning** — Plot routes on interactive maps with drag-and-drop waypoints and real-time distance and time estimates.
- **Smart Itinerary Builder** — Organize stops into day-by-day itineraries with arrival times, durations, and notes.
- **Drag & Drop Organization** — Intuitively reorder stops and days throughout your entire trip.
- **Trip Collaboration** — Invite friends and family to collaborate on trips, vote on stops, and stay in sync.
- **Saved Locations Library** — Build personal collections of favorite spots, restaurants, campgrounds, and viewpoints.
- **Analytics & Insights** — Track travel stats including miles driven, states visited, and favorite stops over time.
- **Offline Access** — Download trip plans for offline navigation without an internet connection.
- **Project Management** — Organize multiple trips into projects, useful for travel bloggers and tour operators.

## Tech Stack

- **Framework:** Next.js with React and TypeScript
- **Styling:** Tailwind CSS with custom theme variables and dark mode support
- **Maps:** Leaflet and React Leaflet for interactive mapping
- **Drag & Drop:** dnd-kit for intuitive trip organization
- **State Management:** Zustand

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## Deployment

This project is deployed on [Netlify](https://www.netlify.com). Pushes to the main branch are automatically built and deployed.
